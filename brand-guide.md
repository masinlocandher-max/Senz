# Masinloc Archive Brand Guide

This guide keeps the Masinloc Archive consistent as the website grows.

## Project Name

Use: **Masinloc Archive**

Avoid changing the project name unless the archive team decides to rename it.

## Voice

The writing should feel:

- respectful
- careful
- community-centered
- clear for students, elders, researchers, and families
- honest about what is verified and what still needs research

## Writing Rules

- Use simple language.
- Explain archive terms when needed.
- Do not exaggerate.
- Do not invent facts to make the site feel complete.
- Mark uncertain information as **oral tradition**, **needs review**, or **research gap**.

## Visual Feeling

The site should feel like a cultural archive, not a tourism brochure.

Good directions:

- archival paper
- coastal town memory
- careful record keeping
- warm community contribution
- readable dictionary and history tools

Avoid for now:

- Destinations sections
- People profile sections
- flashy travel marketing language
- unsupported historical claims

## Current Website Colors

The existing website already uses warm cream, navy, blue, teal, orange, yellow, red, and green. Future updates should build from those colors instead of replacing the whole identity.

## Logo and Name Usage

Keep the logo simple and readable. The name **Masinloc Archive** should remain visible in the navigation or hero area.
